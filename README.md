# Lendr

**Rent out everyday items.**

Lendr is a web application that allows users to rent everyday items from other users or list their own items for others to rent. Users can create an account, browse available listings, view item details, book rentals, and manage listings. Admin users can also view database activity through an admin dashboard.

---

## Features

* User sign up, sign in, and email verification for secure access
* Browse available rental listings uploaded by other users
* Search and filter listings by item name, category, price, location, and condition
* View detailed listing pages with photos, descriptions, reviews, and pickup location
* Create listings with item details, pricing, photos, contact info, and payment preferences
* Edit or delete personal listings through account settings
* Book available items for a selected rental period
* Allow lenders to accept or deny rental requests
* Track personal listings, rental history, and booking requests
* Admin dashboard for viewing database transaction logs

## Purpose

Lendr was created to make renting everyday items easier. Instead of buying items that may only be needed once or for a short period of time, users can rent from others nearby. Lenders can also make use of items they already own by listing them for others to rent.
